Curious Bikes is a web application which displays occupancy and weather information and provides route planning
services for Dublin Bikes to its users.

This project was developed by Group 20:
- Aine Ni Fhlanagain (aine.nifhlanagain@ucdconnect.ie)
- David Keenan (david.keenan1@ucdconnect.ie)
- Eugene Egan (eugene.egan1@ucdconnect.ie)

The running application can be found at:
ec2-107-23-104-194.compute-1.amazonaws.com

This repository contains several python web scraper files and a folder for the application implemented using the python
Flask framework.
